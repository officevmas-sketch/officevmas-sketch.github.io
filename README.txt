VMAS Workplace Full SaaS Web App / iOS PWA

Includes:
- Company login
- Employee list
- Intercom call alert
- Walkie-talkie voice notes
- Punch In / Punch Out
- Attendance logs
- CSV export
- Call and voice logs
- Company Admin dashboard
- Super Admin dashboard
- Plan/user-limit control
- Razorpay demo upgrade placeholder
- Logout
- iPhone PWA support

Setup:
1. Upload all files to GitHub Pages root folder.
2. Edit firebase-config.js with Firebase Web App credentials.
3. In Firebase Realtime Database, import firebase_demo_data.json.
4. For testing, use firebase_rules_testing.json.
5. Open GitHub Pages URL on iPhone Safari.
6. Tap Share > Add to Home Screen.
